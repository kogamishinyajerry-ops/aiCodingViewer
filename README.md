## 📖 项目简介

aiHolmes 是一套完整的 AI 侦探和法律维权工具包，包含：

- **AI 侦探系统** - 智能法律分析和推理
- **职业碰瓷维权工具** - 专业的碰瓷案件分析和管理
- **AI 助手系统** - 智能对话和辅助
- **Skills 和 Agents 系统** - 可扩展的 AI 能力框架

---

## 🎯 核心功能

### 1. AI 侦探系统

一个完整的法律推理引擎，支持：

- ✅ 深度法律推理
- ✅ 事实要素智能提取
- ✅ 争议焦点精准识别
- ✅ 证据缺口深度分析
- ✅ 调查计划自动生成
- ✅ 诉讼策略专业建议
- ✅ 风险评估准确判断
- ✅ 网络侵权案件支持

**总体评分**: 67/100

**适用场景**:
- ✅ 房屋租赁纠纷 (92/100)
- ✅ 简单名誉权侵权 (88/100)
- ✅ 简单肖像权侵权 (85/100)

### 2. 职业碰瓷维权工具

专业的职业碰瓷案件分析和管理系统，包括：

- 14章深度分析，12,000字
- 行动路线分析
- 关键言语举动识别
- 举一反三的深度分析

- 50+项关键证据清单
- 三轨并行诉讼策略

#### 智能工具
- ✅ 行动检查清单生成器
- ✅ 言语举动分析工具
- ✅ 案件管理系统
- ✅ 胜算评估报告

**胜算评估**: 78/100（较有胜算）

**预期补偿**: 20,000-120,000元

## 🚀 快速开始

### 安装依赖

```bash
cd /workspace
pip install -r requirements.txt
```

### AI 侦探系统

```bash
cd ai_detective
./start.sh
```

### 职业碰瓷维权工具

```bash
cd ai_detective
python3 action_checklist_generator.py
python3 speech_behavior_analyzer.py
python3 victimization_case_manager.py
```

### AI 助手

```bash
python3 ai_assistant.py
```
