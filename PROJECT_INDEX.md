# 项目索引

本项目包含两个主要系统：

## 🕵️ AI Detective
> **AI侦探系统** - 用于案件分析和证据收集
> 
> **仓库**: https://github.com/kogamishinyajerry-ops/aiHolmes
> 
### 功能
- 深度推理（Deep Reasoner）
- 证据管理（Evidence Manager）
- 案件分析（Case Manager）
- 时间序列分析
- 言语举动分析

### 文档
- [使用指南](ai_detective/USER_GUIDE.md)
- [项目总结](AI_DETECTIVE_SUMMARY.md)
- [职业碰瓷维权](ai_detective/VICTIMIZATION_INDEX.md)

## 🤖 AI Coding Monitoror
> **AI编程监视官** - 监控AI coding过程，防止记忆丢失和功能破坏
> 
> **仓库**: https://github.com/kogamishinyajerry-ops/aiCodingViewer
> 
### 功能
- 代码变更检测（AST分析）
- 上下文记忆管理
- 回归检测（功能破坏）
- 语义漂移检测
- 实时告警系统
- 代码验证

### 文档
- [使用指南](ai_coding_monitoror/README.md)
- [调研报告](AI_CODING_MONITOROR_RESEARCH.md)
- [交付总结](AI_CODING_MONITOROR_DELIVERY_SUMMARY.md)

### 快速开始
```bash
cd ai_coding_monitoror
./start.sh
# 访问 http://localhost:8001
```
